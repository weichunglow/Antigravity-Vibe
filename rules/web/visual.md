# 🎨 Web Visual System

This document outlines the standard visual system for web applications.

---

## 🎨 Visual System

1.  **Strict Color Usage**: Always use HSL for primary, secondary, and accent colors. Never use `red`, `blue`, or other generic presets.
2.  **Glassmorphism Effects**: Use `backdrop-filter: blur()` combined with semi-transparent backgrounds for cards.
3.  **Vibrant Gradients**: Use linear gradients with high-contrast, harmonious hue shifts.
