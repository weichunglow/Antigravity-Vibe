# 🤝 Contributing to AntiGravity-Vibe

Thank you for your interest in contributing to **AntiGravity-Vibe**! We welcome improvements to existing skills, new reproducible workflows, and aesthetic upgrades.

---

## 🚦 How to Contribute

### 1. Fork the Repository
Fork this repo to your personal GitHub account and clone it locally.

### 2. Create a Feature Branch
Use a descriptive branch name for your changes.

```bash
git checkout -b feature/new-fancy-skill
```

### 3. Add your Skill or Workflow
-   Place skills in `.agents/skills/[skill-name]/SKILL.md`.
-   Place workflows in `.agents/workflows/[workflow-name].md`.
-   Ensure you take advantage of **GitHub-style alerts** (`> [!NOTE]`, `> [!TIP]`, etc.) for readability.

---

## 🎨 Style Guidelines

We prioritize **Rich Visual Aesthetics** over generic code.
-   Use **HSL** for colors in any web demos.
-   Include **Mermaid diagrams** for architecture flows.
-   Ensure no sensitive data or absolute personal paths are committed!

---

> [!CAUTION]
> Every Pull Request will go through automated security scans using our standard `security-audit.md` workflow. Ensure your code passes before submitting!
